# Student Performance Analytics Dashboard

## Problem Statement
Institutions need early warnings to identify students who may fail or drop out.

## Objective
Analyze student data to highlight performance trends and risk areas.

## Dataset
- Marks
- Attendance
- Logins

## Analysis Performed
- Summary statistics
- Correlation analysis
- Identification of at-risk students
- Visualization using bar charts and heatmaps

## Outputs
- Correlation heatmap
- Performance bar chart
- List of at-risk students

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

pip install -r requirements.txt
